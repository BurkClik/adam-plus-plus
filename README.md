# adam-plus-plus

Adam gibi değil, adam++ bir Discord botu.

## Komutlar
| Komut | Açıklama | 
| - | - |
| +adamm @kullaniciadi | Etiketlenen kullanıcının adamlığını över |
| +cuma | Cumanızı kutlar |
| +yüzde | Bilimsel hesaplamalar yaparak adamlık yüzdenizi söyler |

## Özellikler
- 32 adet adamlık sözü
- 12 adet cuma mesajı
